# HTML

- 概念：HTML是一种描述网页的标记语言。

  - HTML是一种超文本标记语言「Hyper Text Markup Language」；
  - HTML不是一种编程语言，只是一种标记语言；
  - 所谓「标记语言」：本质是一套标记标签；
  - HTML使用标记标签描述网页；
  - HTML的标记标签简称HTML标签；
  - HTML是一种解释型语言，由浏览器解释执行。

- HTML文档的基本结构：
  - html：根标签；主要包含2大部分：
    - head：定义文档的头部；包含：meta | title | link | style | script；
    - body：定义文档的主体；其内容会被浏览器显示。
